# Text Reverse Tool

A simple Python program that reverses any text you enter.  
This project is a beginner-friendly console application built to practice loops, strings, and user input in Python.

---

## Features

- Reverse any text instantly
- Simple command-line interface
- Lightweight and fast

---

## 🧠 How It Works

The program reads your input text and reverses it character by character using a loop.

Example:

Input: hello
Output: olleh
